소스코드 해석 : 

CameraMove.cs :

GameOver 당했을 때와 플레이 중 일 때의 카메라의 움직임을 담당하는 스크립트입니다.


Combo.cs : 

적을 타격하거나, 캐릭터가 공중에 떻을때 추가점수를 얻게 되는데, 해당 상황에 콤보 UI를 표출해주는
스크립트입니다.


CommunicateWithServer.cs : 

AWS에 등록되어있는 서버와의 데이터 통신을 담당합니다.
Facebook으로 로그인 할 시, 해당 유저의 토큰값과 플레이 시작시간, 플레이 시간, 플레이 종료시간등을 
JSON형태로 저장하여 mysql 5.7 버전의 Database에 저장합니다.

ConnectToRoom.cs : 

PhotonCloude 서버가와 클라이언트가 통신하여 서버에 등록되어 있는 방에 커넥트 할 술 있도록 해주는 
스크립트입니다.


CustomRoom.cs : 

방에 대한 정보( 방 이름, 최대 참여자 수 )를 저장하는 스크립트입니다.


ExitScript.cs :

Robby에서 종료버튼을 누르면 앱을 종료 할 수 있도록 해주는 스크립트입니다.


FBHolder.cs : 

Facebook과의 연동을 도와주는 스크립트를 도와줍니다. (Facebook 로그인 기능과 FeedShare 기능이 구현되어있습니다.)


GameOVer.cs :

게임이 종료되었을 때 실행될 스크립트입니다.



GM.cs : 

GameManager의 약자로 게임의 진행상태, 게임의 플레이에 필요한 요소를 관리하는 스크립트입니다.



InGameGUI.cs :

모바일 환경에 게임인만큼, 멀티터치와 조이스틱을 관리하는 스크립트 영역입니다(캐릭터 머리위에 뜨는 닉네임도 
위 스크립트에서 관리됩니다)



InRoom.cs : 

게임 내에서 PhotonCloud 서버에 방이 추가되면 실행될 함수가 저장되어있으며, 플레이어를 서버상에 생성시켜주는
역할을 합니다.


Move.cs : 

본 게임은 PC와도 온라인 연동이 가능합니다. 따라서, PC에서도 조작 할 수 있도록 짜여진 코드가 위 코드에 담겨
있습니다.


RobbyUIController.cs :

로비에서 각 UI들이 상호작용 할 수 있도록 함수들이 담겨있는 스크립트입니다.
(게임, 인벤토리, 프렌즈 탭을 통한 액션이 담겨있습니다.)



SelectedCar.cs : 

사용자가 플레이를 할 때 사용할 자동차에 대한 정보가 담기는 스크립트입니다.



UserDataClass.cs :

서버로 부터 받아들인 json 데이터가 파싱되어 저장될 변수들이 정의되어있는 스크립트입니다.
